##### Find Hidden Web Directories - Fuzz URLs with ffuf
- ffuf = työkalu, joka etsii sivustolta piilotettuja kansioita kokeilemalla tuhansia URL-osoitteita automaattisesti
- Tarvitaan myös sanalista, esim. SecLists-projektin **common.txt**
- Peruskomento: `ffuf -w common.txt -u http://127.0.0.2:8000/FUZZ` -> sana FUZZ korvataan vuorollaan jokaisella sanalistan rivillä
-  ffuf on nopea ja tekee paljon pyyntöjä -> käytä vain omiin tai luvallisiin kohteisiin, harjoitellessa kannattaa irrottaa netti

##### ffuf - Fuzz Faster U Fool - README.md
- ffuf = "Fuzz Faster U Fool" — Go-kielellä kirjoitettu nopea web-fuzzer, jonka luonut Joona Hoikkala (joohoi)
- Avainsana FUZZ sijoitetaan kohtaan, jota halutaan fuzzata — voi olla URL:ssä (-u), headerissa (-H) tai POST-datassa (-d)
- Käyttötapauksia:
  - Hakemistojen etsintä: `ffuf -w wordlist.txt -u https://target/FUZZ`
  - Virtuaalihostien löytäminen: `-H "Host: FUZZ"`
  - GET-parametrien nimet/arvot: `?FUZZ=test_value tai ?nimi=FUZZ`
  - POST-loginit: `-d "username=admin&password=FUZZ"`
- `-mc` (matchaa statuskoodit), `-fc/-fs/-fw/-fl` (suodatus), `-ac` (autokalibrointi), `-recursion`,`-maxtime`, `-o` (tulosten tallennus json/html/csv)
- Asetukset voi tallentaa pysyvästi ~/.ffufrc-tiedostoon
- README painottaa, että ffuf ei ole pelkkä "dirbuster" — sama työkalu sopii niin SQL-injektioiden, kirjautumisen brute-forcen kuin headerien testaamiseen


ffuf on jo minulle hieman tutumpi työkalu ja tätä on käytetty sovellusten hakkerointi kurssilla.

#### a) Fuzzzz
Latasin ensin ffuf:n komennolla `sudo apt-get install ffuf`, mutta se olikin ladattu koneelle valmiiksi. Seuraavaksi latasin/hain sanalistan komennolla `wget https://raw.githubusercontent.com/danielmiessler/SecLists/master/Discovery/Web-Content/common.txt`. Sitten hain/latasin dirfutz-1:n komennolla `wget https://terokarvinen.com/2023/fuzz-urls-find-hidden-directories/dirfuzt-1`. Ajoin viellä komennon `chmod u+x dirfuzt-1` oikeuksien takia ja `./dirfuzt-1` avatakseni sivun:

#### b) Fuff me
Asensin ensin docker.io komennolla `sudo apt-get install docker.io git ffuf`. Sitten kloonasin FuffMe-harjoitusmaalin ja rakensin Docker-kontin komennolla `git clone https://github.com/adamtlangley/ffufme`, `cd ffufme/` ja `sudo docker build -t ffufme .`. Käynnistin kontin porttiin 80 komennolla `sudo docker run -d -p 80:80 ffufme`

<img width="1292" height="1160" alt="image" src="https://github.com/user-attachments/assets/582f12d8-f063-449a-b7f1-5232a71534e0" />

Kuvassa näkyy myös testaus, että maali toimii. Tein sen komennolla `curl -si localhost | grep title`, joka tulosti **<title>FFUF.me</title>**. Avasin sivun myös selaimella osoitteessa http://localhost/:

`mkdir ~/Downloads/wordlists`, `cd ~/Downloads/wordlists`, `wget http://ffuf.me/wordlist/common.txt`, `wget http://ffuf.me/wordlist/parameters.txt`, `wget http://ffuf.me/wordlist/subdomains.txt`.
